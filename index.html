The Puzzler

The Puzzler is an immersive mobile VR application that uses “Simon Says” game mechanic. The game is set in a medieval dungeon which sits in a remote mountain valley. As the player hears the plaintive sounds of Scottish bagpipes, she is invited to enter the dungeon and solve the puzzle.
